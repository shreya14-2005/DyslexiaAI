# DyslexiaAI

DyslexiaAI is an AI-powered assistant designed to support dyslexic students with reading, writing, and learning tasks. It combines a backend language model and a frontend interface to make learning more accessible.

## Features
- Suggests simpler words and sentence structures
- Helps identify difficult words and provides explanations
- Offers an easy-to-use, dyslexia-friendly interface

## Project Structure
- `backend/`: Python backend, model training, and data processing
- `frontend/`: Web UI built with Vite and React

## Getting Started
1. Install backend dependencies in `backend/`
2. Install frontend dependencies in `frontend/`
3. Run the backend and frontend together for the complete app experience
